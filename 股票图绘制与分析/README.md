## 01.绘制某只股票的收盘价折线图和k线图

### 步骤

1. 使用Numpy读取文件，并将时间字符串转为规定格式的字符串
2. 使用Matplotlib进行可视化，x轴为时间，y轴为价格
3. 设置刻度定位器，x轴主刻度为每周一，次刻度为每天
4. 绘制收盘价折线图
5. 绘制蜡烛图
6. 绘制k线图
